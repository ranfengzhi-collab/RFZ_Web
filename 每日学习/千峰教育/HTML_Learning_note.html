HTML学习笔记

HTML(Hyper Text Markup language)是什么？

HTML是一种超文本标语言 （注意！！！  HTML不是编程语言，是用来描述网页的语言）

万维网(world wide web consortium)(W3C)

web网页的组成
  HTML:属于结构
  CSS：表现方式
  JavaScript:行为

HTML的基本文本语言
双标记: <标记  属性=“属性值”>  </标记> 单标记: <标记  属性=“属性值”>

文本标题(heading)(文本标题最多到六级标题.h1~h6)

<h2>   </h2>二级标题
<h3>   </h3>三级标题
<h4>   </h4>四级标题
<h5>   </h5>五级标题
<h6>   </h6>六级标题

span跨度

列表
  1. 无序列表

    2. <ul>
      <li> </li>
    </ul>
    无序列表的样式默认是黑色实心原点
    • 黑色实心圆点

    通过更改<type="属性值">
    将属性值改为 circle为空心圈
    • 空心圆点

    将属性值改为square为实心正方向
    • 实心正方形

    将属性值改为none 为取消显示项目编号
  2. 有序列表

    3. <ol>
      <li> </li>
    </ol>
    有序列表是有序号的,序号为a,A,i(罗马数字小写),I(罗马数字大写),1(阿拉伯数字)
    1. 序号为A
    2. 序号为B

    1. 序号为a
    2. 序号为b

    1. 序号为i
    2. 序号为ii

    1. 序号为I
    2. 序号为II

    1. 序号为1
    2. 序号为2

    <ol type="A"或者"a"或者"i"或者"I"或者1> </ol>即可按照序列编号
    将有序列表的序号按照指定位置进行排序
    <type="A" start="3">即可实现从C开始编号
    1. 序号C

  3. 自定义列表

    4. <dl>
      <dt> </dt>
      <dd> </dd>
    </dl> 自定义列表一般用于图文显示当中,<dd>中为图片;<dt>中为文字;
    自定义列表的dt文字部分
    自定义文字列表的dl图片部分
